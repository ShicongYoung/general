# TV 版 2026年3月活跃工厂数

## 查询时间
2026年3月整月（2026-03-01 ~ 2026-03-31）

## 数据来源
- **数据库**：小工单_阿里云_prod_starrocks
- **表**：`trace_log_dp`
- **实例**：StarRocks

## 口径说明

### 活跃工厂（核心指标）
定义：当月 `event='tv-device-info'` 埋点中，出现 **≥2 个不同自然日** 的去重工厂数（`orgId`）

```sql
SELECT COUNT(*) FROM (
  SELECT orgId
  FROM trace_log_dp
  WHERE dat BETWEEN '2026-03-01' AND '2026-03-31'
    AND event = 'tv-device-info'
  GROUP BY orgId
  HAVING COUNT(DISTINCT dat) >= 2
) x
```

### 任意出现（参考指标）
定义：当月 `event='tv-device-info'` 有任意埋点记录的去重工厂数

```sql
SELECT COUNT(DISTINCT orgId)
FROM trace_log_dp
WHERE dat BETWEEN '2026-03-01' AND '2026-03-31'
  AND event = 'tv-device-info'
```

## 查询结果

| 指标 | 工厂数 |
|------|--------|
| **活跃工厂（≥2天）** | **757** |
| 任意出现（≥1天） | 984 |
